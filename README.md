# Сайт за компютърни части

Курсов проект - Уеб приложение за управление на компютърни части.

## Технологии

- HTML5
- CSS3
- JavaScript (jQuery)
- PHP
- MySQL

## Функционалност

Приложението включва следните операции за работа с базата данни:

- **INSERT** - Добавяне на нови продукти
- **UPDATE** - Редактиране на съществуващи продукти
- **DELETE** - Изтриване на продукти
- **SELECT** - Извличане и визуализиране на всички продукти
- **SELECT WHERE** - Търсене и филтриране на продукти по различни критерии

## Инсталация

### 1. Импортиране на базата данни

Отворете phpMyAdmin или MySQL конзолата и изпълнете файла `sql/database.sql` за създаване на базата данни и таблиците.

### 2. Конфигуриране на базата данни

Редактирайте файла `config/database.php` и задайте правилните параметри за вашата база данни:

```php
define('DB_HOST', 'localhost');
define('DB_USER', 'your_username');
define('DB_PASS', 'your_password');
define('DB_NAME', 'computer_parts_db');
```

### 3. Запускане на проекта

Поставете файловете в директорията на вашия уеб сървър (например `htdocs` за XAMPP или `www` за WAMP).

Отворете браузъра и навигирайте до: `http://localhost/survurnoprogramirane/`

## Защита

Приложението включва защита срещу:

- **SQL Injection** - Използване на prepared statements
- **XSS атаки** - Екраниране на входни данни с `htmlspecialchars()`
- **Валидация** - Клиентска и сървърна валидация на данните
